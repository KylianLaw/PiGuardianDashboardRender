About PiGuardian

A smart home monitoring system powered by Raspberry Pi, Adafruit IO, PostgreSQL (Neon), and a Flask web dashboard.

Project Overview

PiGuardian is a complete IoT ecosystem designed to monitor your home’s environment and security in real time. It collects temperature, humidity, and motion information from the Raspberry Pi, stores the data securely in a cloud PostgreSQL database, and visualizes it through an interactive Flask dashboard.

Users can view historical environmental charts, monitor live sensor data, review intrusion logs, and remotely control IoT devices such as LEDs, a buzzer alarm, and an LCD display. The system integrates cloud services (Adafruit IO and Neon DB) and is fully deployed on Render.com for reliable access anywhere.
